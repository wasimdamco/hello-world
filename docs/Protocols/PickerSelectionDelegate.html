protocol PickerSelectionDelegate

protocol: PickerSelectionDelegate : the protocol function to notify back the class or model or controller who implements and sets PickerSelectionDelegate to self about what is selected from picker.

  • didFinishSelectingRow(_:dataSource:input:)
